# Daily Reflection Tree - Design Explanation

## Approach
I designed a structured reflection system focusing on three psychological axes:
1. Locus of control (responsibility)
2. Contribution vs entitlement
3. Awareness beyond self

## Design Choices
Each question has fixed options to ensure deterministic behavior.
Decision nodes guide the flow based on user choices.
Reflections are written to encourage thinking without judgment.

## Structure
The flow starts with general reflection, then moves through:
- Personal responsibility
- Contribution to others
- Awareness of broader impact

## Improvements
With more time, I would:
- Add deeper branching
- Personalize summaries more
- Improve psychological depth
